# Zora Cohen - Professional Portfolio

A modern, professional portfolio website showcasing multi-disciplinary entrepreneurial work across product design, web development, photography, and event production.

## 🎯 Purpose

This site positions Zora Cohen as a serious multi-disciplinary entrepreneur, emphasizing capabilities over biography. It serves as:
- A professional portfolio for freelance and full-time opportunities
- A showcase of technical competency using modern web technologies
- A platform to demonstrate range: products, development, photography, events

## 🏗️ Built With

- **[Astro](https://astro.build)** - Static site framework
- **[React](https://react.dev)** - Interactive components
- **[shadcn/ui](https://ui.shadcn.com)** - UI component library
- **[Tailwind CSS](https://tailwindcss.com)** - Styling
- **[TypeScript](https://www.typescriptlang.org/)** - Type safety

## 📂 Project Structure

```
├── src/
```

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ or Bun
- npm, yarn, pnpm, or bun

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The site will be available at `http://localhost:4321`

- **Primary (Green)**: `hsl(142, 76%, 36%)` - Growth, innovation, sustainability
